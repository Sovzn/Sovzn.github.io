Linux进程管理

2021-01-27
Linux-Linux

进程管理

进程基本概念

1.在Linux中，每一个程序都是有自己的一个进程，每一个进程都有一个id号。

2.每一个进程都有一个父进程。

3.进程有两种存在方式：前台！后天运行。

4.一般的话服务都是后台运行的，基本的程序都是天台运行的。

命令

查看进程

ps： 查看当前系统中正在执行的各种进程的信息

参数：

  • -a 显示当前终端运行的所有进程的信息、

  • -u 以当前用户的信息显示进程

  • -x 显示后台运行进程的参数

# ps -aux 查看所有的进程
# | 为管道符    grep：查找文件中符号条件的字符串.
# ps -aux|grep mysql  查看mysql相关的进程

ps -ef：可以查看父进程

也可以通过进程树来查看父进程:

pstree

​ -p 显示父id

​ -u 显示用户组

终止进程

kill -9  进程id  #强制终止进程，一般不要随便终止进程
